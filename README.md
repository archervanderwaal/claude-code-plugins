# Minimalist Plugins

Minimalist plugins for Claude Code.

## Installation

```bash
/plugin marketplace add https://github.com/0xarcher/claude-code-plugins.git
```

Then restart Claude Code.

## Available Plugins

| Plugin | Description | Category | Commands |
|--------|-------------|----------|----------|
| [todo](./plugins/todo) | Minimalist todo list with pure text output. Tasks stored in `~/.claude/TODO.md` | Productivity | `/todo:list`, `/todo:add`, `/todo:complete`, `/todo:delete`, `/todo:edit`, `/todo:clear` |
